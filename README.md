##  About the Project

**Sulfad Stitches** is a dedicated fashion and tailoring website that bridges the gap between traditional craftsmanship and modern digital convenience. The platform serves as a virtual showroom displaying bespoke designs, fashion lookbooks, and available services to help clients achieve their perfect fit.

> _Elegance is not about being noticed, it's about being remembered._

---

##  Features

- **Interactive Lookbook** — Dynamic display of fashion collections and custom designs
- **Service Catalog** — Detailed breakdown of tailoring, styling, and alteration services
- **Fully Responsive** — Optimized across mobile, tablet, and desktop devices
- **Modern UI/UX** — Clean, elegant design tailored to a premium fashion brand
- **Contact & Booking** — Easy navigation for client inquiries and appointment booking

##  Getting Started

### Prerequisites

Ensure you have **Node.js** installed on your machine.
👉 [Download Node.js](https://nodejs.org/)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/sulfad-stitches.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd sulfad-stitches
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the development server**

   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000`
